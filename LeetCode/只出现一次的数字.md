
给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:
```
输入: [2,2,1]
输出: 1
```
示例 2:
```
输入: [4,1,2,1,2]
输出: 4
```


解决方法：
可以使用异或

1.恒定律：A ^ 0 = A
2.归零率：A ^ A = 0
3.交换律：A ^ B = B ^ A
4.结合律：(A ^ B) ^ C = A ^ (B ^ C)

题目可知，除一个单个数字外，其余都是成对出现的

代码：
```
int singleNumber(int* nums, int numsSize) 
{
	int i=0;
	int result = 0;
	for(; i<numsSize; i++)
	{
		result ^= nums[i];
	}
	return result;
}
```
