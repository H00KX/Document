# XPath

XPath是一门在XML文档中查找信息的语言。XPath用于在XML文档中通过元素和属性进行导航。

# XPath 路径表达式

XPath使用路径表达式来选取XML文档中的节点或者节点集。

在XPath中，有七种类型的节点：

- 元素
- 属性
- 文本
- 命名空间
- 处理指令
- 注释
- 文档节点
	
# XPath 术语

```
<?xml version="1.0" encoding="ISO-8859-1"?>

<bookstore>

<book>
  <title lang="en">Harry Potter</title>
    <author>J K. Rowling</author> 
	  <year>2005</year>
  <price>29.99</price>
</book>

</bookstore>
```
